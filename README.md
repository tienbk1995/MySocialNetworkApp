# MySocialNetworkApp

A full-stack social networking prototype built with the **MERN** stack (MongoDB, Express, React, Node).  
This project demonstrates end-to-end web development skills: API design, real-time features, authentication, responsive UI, and deployment. It complements my embedded systems background by showing full-stack engineering capability and product thinking.
